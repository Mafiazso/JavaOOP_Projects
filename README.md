UI : set ภาษาไทย
Font : Tahoma
- เปลี่ยนชื่อโปรแกรมจาก PomodoroApp -> TaskMaster
- เพิ่มส่วนของ UIManager (บรรทัดที่ 92-107) 
เพื่อให้ภาษาไทยแสดงผล หากไม่ได้ใส่เข้าไป บางเครื่องที่รันอาจจะไม่แสดงภาษาไทย จึงเขียนเพื่อกัน error
- เพิ่ม บรรทัดที่ 126-128 
border.setTitleFont(new Font("Tahoma", Font.PLAIN, 14)); 
set ค่าภาษาไทย เพราะไม่สามารถทำกับ UIManager ได้ เนื่องจากทำได้เพียงแค่ component แต่ส่วนที่เพิ่มเข้าไป มันเป็นของ border
