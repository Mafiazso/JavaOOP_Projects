Pomodoro Timer
- มีเวลาทำงานทั้งหมด โดยจะอัปเดททุกๆวินาทีที่ทำงาน และเก็บลง DB
- เมื่อทำงานเสร็จแล้วจะมีเวลาพักเบรค
- สามารถตั้งค่า Pomodoro ได้ที่หน้าเมนู Setting
    - ตั้งค่าเวลาแบบ 25/5 , 50/10 ทำงานก็ต่อมาเมื่อสร้างงานใหม่ หรือ กด reset 
    - ตั้งค่าแบบ custom เอง กด Save เพื่อยืนยัน
    - มีเปิด-ปิดแจ้งเตือนเมื่อ Pomodoro หมดเวลา

Setting Menu
- ทำระบบ Setting ภายในโปรแกรมสำหรับ Pomodoro Timer , Notification

Export File
- เมื่อกดส่งออกในหน้า Task Memu งานทุกงาน (งานปกติและงานขยะ) จะถูกเขียนไว้ในไฟล์ exported_tasks.txt ภายในโฟลเดอร์ Project 

สิ่งที่แก้
- แก้ภาษาไทย และ การแสดงผลของ Emoji
    - Emoji Font : Segoe UI Emoji
    - Thai Font : Tahoma
    - ลบ Emoji ออกในบางจุดเพื่อให้แสดงผลให้ดีที่สุด
    - (ที่ต้องใช้แยกกันเพราะ font ไทย และ อิโมจิมักจะไม่สามารถแสดงผลด้วยกันได้)
